# Dashboard

O dashboard interativo foi desenvolvido no Looker Studio e pode ser acessado pelo link abaixo:

🔗 https://datastudio.google.com/reporting/19bd09e1-e599-43b6-9527-e60f18ab9a18

## Visão geral

Este dashboard apresenta indicadores relacionados à disponibilidade de recursos de acessibilidade em títulos da Netflix e Disney+, incluindo:

- Closed Caption (CC)
- SDH
- Audiodescrição (AD)
- Legendas simples em português

Também permite comparar as plataformas e explorar os dados por meio de visualizações interativas.
